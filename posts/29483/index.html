2023年的上古卷轴5个人向mod推荐

发表于 2023-01-11 更新于 2023-03-31

前言

都2023年辣 居然还有人在玩2011年（其实是2016年）的游戏吗
经久不衰的上古卷轴5即使到了2023年仍然有一堆玩家在玩（我写的时候到steamdb看了下还有16k的特别版玩家和1.7k的传奇版玩家在玩）
话说这其实是b社在滚6上摆烂的结果吧（btw：总感觉星空会烂 杯赛现在真的有产能做东西吗）
但一个2011年的游戏放到现在 怎么说呢 要是看不出来一堆问题那游戏界没救了 比如我最讨厌的打击感过差和甚至需要mod来修的一堆bug
总而言之 滚5需要mod来进行完善 可我发现（也许是我的搜索不过关的问题） 现在的mod推荐有亿点过时 而且整合包一大堆
于是就打算写个mod推荐 和上一篇一样是边装边写

ps：没有实验室 这玩意过于头疼 根本不想碰

个人配置

5600g + 16g ddr4 3200mhz + rx580 4g

没有独显 因为我穷 所以这篇文章对于画质可能不会怎么提及（个人向嘛）

游戏版本是1.6.640.0 十周年纪念版 英文版（官方繁中版本似乎落后？我没测试过）

在Windows 11上运行

Mod推荐

前置工作

  • 游戏本体（废话）

  • Skyrim Script Extender (SKSE) 其实可以不装 但绝大多数mod都要依靠这个

  • mod启动器/管理器（或者别的名字？）
    主流启动器大概有三个 Nexus Mod Manager（简称NMM（已经过时到根本不更新了）） Mod Organizer 2（简称MO2） Vortex（NMM的精神续作？） 本文使用mo2 因为我喜欢（还是想试试Vortex怎么办（（

基本类

  • 汉！化！要是想玩英文版当我没说（（
    重光汉化 如果想找什么别的版本的汉化那也行 反正只是名词翻译不同

  • 非官方修正补丁（Unofficial Skyrim Special Edition Patch）和 ANK汉化 或 重光汉化版本
    但是有个问题就是mod制作组会改掉一切他们认为的不合理的地方 所以其实也算是修改了游戏性 装的时候考虑一下？

  • 创意模组非官方补丁（Unofficial Skyrim Creation Club Content Patches）和 ANK汉化 或 汤镬汉化版本（如果用重光那就用这个）
    问题同上 这个如果没买十周年dlc也得装 区别只是没买的选Individual 买了的选Merged（似乎是的 英文看不懂啦.jpg）

到这里进游戏应该有汉化 如果想体验原版那就到这了 但后续也许有一些增强体验的？

  • SkyUI 和 汉化

  • Address Library for SKSE Plugins

  • MCM Helper 汉化还是上面的上面那个 有两个文件 安装略麻烦

  • RaceMenu 和 汉化 注意这个汉化别下错了
    捏脸界面 但是是一些mod的前置？ 同位替代ECE 但我喜欢rm（（

  • FNIS动作框架 好像有别的动作框架来着？
    或Nemesis
    注意两个动作框架冲突（好像有解决方法）若无动作mod需求则别装动作框架

  • 骨骼框架（XP32 Maximum Skeleton Special Extended - XPMSSE）和 汉化
    说实话这个和上面的捏脸应不应该放在这里我不好说 因为挺重要但不玩人物建模改善之类那似乎也没那么重要？

  • PapyrusUtil SE
    差点忘了这个

补丁类

大概是那种打上不会有问题的 单纯改善游戏体验的（当然出问题我也不知道怎么解决

  • Better Dialogue Controls
    玩家帮b社修bug+1 因为b社的判定问题 原版的对话时常出现选择的和鼠标选中的不一样的情况 这个mod修复了该问题

  • Joy of Perspective SSE 和 周年版补丁
    第一人称改善 注意上面那个骨骼框架在安装时有这个相关的兼容补丁 另外N网还有这个 似乎后缀有SSE的是新版？

  • 3PCO - 3rd Person Camera Overhaul 和 汉化
    第三人称改善 有点小bug

  • Proper Aiming
    第三人称下的瞄准修正 原版用第三人称瞄准的话会有偏差

  • (SKSE64) Havok Fix
    虽然对我而言没意义但还是得说 b社的物理引擎bug一堆以至于当游戏帧数超过60会有亿堆bug包括但不限于开头马车疯狂翻滚 这个mod使游戏在60帧以上运行时bug少点 对我没意义的原因是就算原版最低画质我这电脑跑30帧都危险（笑

  • Realistic Ragdolls and Force SE
    b社的物理引擎烂的出奇 物理效果也是烂的出奇 这个mod使游戏里的物理效果至少合理了点

  • Achievements Mods Enabler
    成就解锁 我是说steam成就 如果没有skse那似乎还需要个前置

  • Modern Brawl Bug Fix
    拳击bug补丁 我真是服了b社

  • SSE Engine Fixes (skse64 plugin)
    引擎修复 你知道我想说什么

  • RaceCompatibility for Skyrim Special Edition
    独立种族补丁 如果不想添加别的种族那无所谓

美化类

我的电脑不行所以什么大规模改动在这篇文章是看不见的

  • A Quality World Map
    地图细化

  • Static Mesh Improvement Mod - SMIM 和 汉化
    静态模型改善 就是给游戏里的一堆东西细化了贴图

  • Enhanced Blood Textures 和 汉化
    优化了血液贴图 聊胜于无吧

  • Enhanced textures detail
    增强纹理细节 看描述似乎不会影响性能？和静态模型那个有覆盖所以顺序要调

  • Realistic Water Two SE 和 汉化（ANK版）或 汉化（汤镬版）
    水体改善 额 不知道会不会影响性能 希望我的机子撑得住

  • Book Covers Skyrim 和 汉化
    书皮 好看就行

  • Immersive Sounds - Compendium 和 汉化
    声效

写到这里我突然发现一件事 Nexus的汉化基本用的都是汤镬的名词啊 ANK挺少见

  • Enhanced Lights and FX 和 汉化
    光照增强

  • 人物美化

    啊啊啊这东西头疼程度在我看来仅次于实验室 可偏偏滚5的建模。。。

    这两种身形其实没什么优劣之分 只是个人喜好问题 但这只是框架 美化还得。。。

功能类

  • Auto Hide Ammo
    自动隐藏弓箭 字面义

  • Unread Books Glow SSE
    未读书籍发光 字面义

  • Jaxonz Positioner Converted 和 繁中翻译
    物品放置增强？滚5原版的物品放置只能以悲剧形容 如果玩辐射4那可能上手这个mod非常简单 原因嘛 b社抄mod了 简中翻译自行寻找 使用繁中汉化注意字库问题

  • A Matter of Time - A HUD clock widget 和 MCM支持 以及 MCM汉化
    钟 能够在游戏内查看游戏时间和现实时间 本体汉化在本体的文件列表

  • Atlas Map Markers SE 和 繁中汉化简中汉化
    地图标记扩充 简中汉化自行寻找 使用繁中注意字库

  • Quick Light SE 和 MCM设置
    增添一个腰间的灯 避免探索的时候还放个照明术或者拿火把

  • SkyHUD
    UI修改 挺好看
